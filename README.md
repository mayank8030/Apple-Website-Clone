# iPhone 17 Pro — The Atelier Edition

A redesigned, luxury-styled concept landing page for the iPhone 17 Pro — built as a front-end clone project using pure **HTML & CSS**.

🔗 **Live Site:** [mayank8030.github.io/Apple-Website-Clone](https://mayank8030.github.io/Apple-Website-Clone/)

---

## About

This project reimagines a typical Apple-style product page with an editorial, boutique-catalog aesthetic — obsidian black backgrounds, champagne gold accents, serif display type paired with clean sans-serif body text, and gold "certificate plaque" styling for spec highlights.

## Features

- Full-bleed autoplay hero video with overlay headline
- Horizontal scroll-snap highlight cards (Design, Chip, Camera, Center Stage, iOS 26, Apple Intelligence)
- Quick-offer strip (Trade-In, Mac, Apple Watch, Student Discount)
- Dedicated sections for Design, Cameras, Performance, and Battery
- Fully responsive — desktop, tablet, and mobile layouts
- Custom typography using [Fraunces](https://fonts.google.com/specimen/Fraunces) (display) and [Inter](https://fonts.google.com/specimen/Inter) (body)

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- Google Fonts

## Running Locally

1. Clone the repository
   ```bash
   git clone https://github.com/mayank8030/Apple-Website-Clone.git
   ```
2. Open `index.html` in your browser — no build step required.
